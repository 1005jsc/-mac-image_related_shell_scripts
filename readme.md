# 맥 이미지 관련 shell scripts

## 1. compress.sh

    이미지 png, jpg를 압축(** 마지막 숫자에 1~9의 숫자를 넣으면 된다, 1이면 덜 압축 9면 강하게 압축)
    커멘드는 .sh파일에 남아 있음

## 2. pdf_to_png.sh

    입력: ./pdf_to_png.sh png 300 GD_png
    # 인자는 총 3개:
    # png로 할꺼냐 jpg로 할꺼냐
    # 해상도 값(300이 가장 표준, 1부터 입력가능)
    # 저장할 파일 이름
    # 예) ./pdf_to_png.sh png 300 GD_png
